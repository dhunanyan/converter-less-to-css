Główne cele i założenia projektu: <br>

Język implementacji: Python <br>

Generator parserów: ANTLR4 <br><br>
Narzędzie ANTLR zostało użyte do realizacji projektu. Lekser oraz generator parsera wchodzą w jego skład.
